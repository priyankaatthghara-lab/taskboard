# 📋 Task Board Website

A simple and interactive Task Board application built using **HTML, CSS, and JavaScript**.  
It helps users manage their daily tasks efficiently by adding, updating, and deleting tasks.

---

## 🚀 Features

- ➕ Add new tasks  
- 📝 Edit existing tasks  
- ❌ Delete tasks  
- 📌 Mark tasks as completed  
- 📱 Responsive design (works on mobile & desktop)  

---

## 🛠️ Technologies Used

- HTML5  
- CSS  
- JavaScript
